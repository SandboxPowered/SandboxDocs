## Core Events

### OnResourceStart

```typescript
onResourceStart(): void
```

An event that is run after a resource has started.  
**Parameters** `none`   
**Returns** `void`  
**Side** `both`

### OnResourceStop

```typescript
onResourceStop(): void
```

An event that is run before a resource is stopped.  
**Parameters** `none`   
**Returns** `void`  
**Side** `both`
